安装及配置 Pelican 博客

Pelican

Pelican 是一种静态博客生成器。

准备工作

为了发布在 GitHub Page 上，需要先新建一个远程仓库 username.github.io

将该仓库克隆到本地

$ git clone https://github.com/{username}/{username}.github.io.git blog

由于 Pelican 默认发布 master 分支下的内容，所以，我们将 pelican 项目放到其他分支中。

$ git checkout -b source

安装

新建 python 虚拟环境

$ cd blog

$ python -m venv .venv

$ source .venv/bin/activate

安装必要的 python 包。

$ python -m pip install pelican markdown ghp-import

新建 pelican 项目

$ pelican-quickstart
